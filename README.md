# AimSAT
A real interactive SAT Dashboard to help SAT students study for the exam and Track their progress. it also help with time management, strengthening of weak sections, and preps for a real SAT. currently we're at stage 1.
=======


## Getting Started

First, run the development server:

```bash
npm run dev

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
